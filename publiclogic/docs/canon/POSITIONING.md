**Status:** canon (positioning frame)
**Date:** 2026-07-11
**Origin:** a passing line — *"so you want to be the Amazon of templates"* — and the answer
*"kinda."* This note keeps the *"kinda."* It pulls the one true thing out of the Amazon

## The principle

**Templates are the complement. Commoditize them. The governed runtime is the product.**

Everyone misremembers Amazon's moat. It was never the catalog — "the everything store" was
the **wedge**. The moat was **fulfillment**: the boring, capital-intensive layer underneath
that no competitor wanted to build. Bezos commoditized the complement (books, then
everything) and owned the layer beneath it (logistics, then AWS).

Apply it exactly:

- A template on Squarespace / CivicPlus / a shared Google Doc is a **dead artifact** — it
  renders and stops.
- The same template *on PuddleJumper* seals, retains, attributes, and **proves**. That is
  fulfillment. It is the thing a template marketplace structurally **cannot** have, because
  it has no runtime.
- So let templates be abundant, forkable, cheap, even free. **Do not defend the catalog —
  defend the spine.** The template is the razor; the governed runtime is the blade.

## The mapping

| Amazon | PJ / MM |
|---|---|
| Seller Central — list once, it compounds | **MM (the authoring layer)** — make a *governed module* once; every town reuses it. The catalog-production engine. |
| Fulfillment / Prime — the real moat | **PJ (the runtime)** — every module, whoever authored it, runs on the same verified spine: sealed, retained, audit-ready. |
| Star reviews | **Receipts.** *"Sealed 4,000 records across 12 towns, zero audit findings"* annihilates a five-star rating. **Proof is the review.** |
| Flywheel: selection → traffic → sellers → selection | more towns → more *proven* modules → more proof → more towns. **The flywheel spins on proof, not price.** |

## The trap (why only "kinda")

If we *position* as "the Amazon of templates," we have re-committed the
**Governance-vs-Government error one layer up**. "Template store" is the commoditized,
dead-artifact framing; a competitor undercuts the catalog next quarter. "The runtime that
makes any template govern and prove itself" is the defended category.

Same discipline as the naming canon: **do not sell the noun everyone can copy; sell the verb
only we can guarantee.** Selection is not a moat. Fulfillment is. For us, fulfillment is
governance + the verified record.
