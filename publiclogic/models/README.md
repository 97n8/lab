# models/ — Core domain models

CaseSpace is the only **top-level primitive**. The others here are its parts and
supporting domain types, not peers.

Planned structure:

- `casespace/` — the governed operating record (standing/closing)
- `form/` · `module/` · `record/` · `thread/`
- `organization/` · `person/` · `asset/` · `location/` · `timeline/`
